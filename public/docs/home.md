# 欢迎使用文档站点

这是一个基于 Vue 2 的文档站点，支持 Markdown 文件加载。

## 功能特点

- 支持加载外部 Markdown 文件
- 实时预览
- 响应式设计
- 左侧导航菜单

## 使用方法

1. 在 `public/docs` 目录下创建 Markdown 文件
2. 在 `src/data/menu.js` 的 `menuItems` 数组中添加对应的菜单项
3. 点击左侧菜单即可查看对应文档
